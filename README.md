# Portfolio

This repository contains my personal portfolio website.

## About

This site presents my projects, skills, and ways to contact me.

## Development

- Install dependencies:
  ```bash
  npm install
  ```
- Start the development server:
  ```bash
  npm run dev
  ```
- Build for production:
  ```bash
  npm run build
  ```
