Про мене

Я — студент ЛНТУ, спеціальність кібербезпека. Портфоліо будується на основі мого GitHub профілю imtiredwhenitalk.

Зараз фокусуюсь на базі для практичної кібербезпеки: мережі (TCP/IP, DNS, HTTP, маршрутизація), Linux-системи (права доступу, процеси, логування), та інструменти з Kali Linux для навчальних лабораторних і CTF.

Вчуся акуратно мислити як аналітик: збирати факти, читати логи, відтворювати проблеми та оформлювати короткі звіти. У репозиторіях нижче — мої навчальні роботи та експерименти.
